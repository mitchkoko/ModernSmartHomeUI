# Modern Smart Home UI

> A clean, modern smart home interface built with Flutter — toggle devices, view status, and control your home with smooth animations.

![Screenshot](https://user-images.githubusercontent.com/29016489/200977277-210b239c-fd92-435e-ad99-0cccd6035afd.JPG)

## Features

- Modern dark-themed smart home dashboard
- Interactive device toggle controls (lights, AC, TV, fan)
- Horizontal scrolling device cards
- Google Fonts integration for clean typography
- Smooth power on/off animations

## Tech Stack

- Flutter & Dart
- Google Fonts

## Getting Started

```bash
git clone https://github.com/mitchkoko/ModernSmartHomeUI.git
cd ModernSmartHomeUI
flutter pub get
flutter run
```
